# Strapi 行业门户解决方案

基于 Strapi v4 的行业门户网站解决方案，提供多个行业的网站模板，帮助企业快速构建专业的门户网站。

## 🎯 主要特点

- 基于 Strapi v4 开发
- 支持多个行业场景
- 模块化设计
- 易于扩展
- 完整的文档
- 开箱即用

## 🚀 快速开始

### 环境要求
```bash
Node.js >= 16
PostgreSQL >= 10
Redis >= 6
```

### 安装步骤
1. 选择行业模板
```bash
npx create-strapi-app my-site --template [industry]
```

2. 安装依赖
```bash
cd my-site
yarn install
```

3. 启动服务
```bash
yarn develop
```
