# 📚 Strapi 教育培训门户模板

## 📋 项目介绍
这是一个基于Strapi v4的教育培训行业门户网站模板，提供完整的教育培训机构网站解决方案。集成课程管理、学员管理、教学管理、在线学习等核心功能，支持快速构建专业的教育培训门户网站。

## 🚀 快速开始
### 环境要求
\`\`\`bash
Node.js >= 16
PostgreSQL >= 10
Redis >= 6
\`\`\`

### 安装步骤
1. 克隆模板
\`\`\`bash
npx create-strapi-app my-training-site --template training
\`\`\`

2. 安装依赖
\`\`\`bash
cd my-training-site
yarn install
\`\`\`

3. 环境配置
\`\`\`bash
cp .env.example .env
# 配置数据库和其他设置
\`\`\`

4. 启动服务
\`\`\`bash
yarn develop
\`\`\`
