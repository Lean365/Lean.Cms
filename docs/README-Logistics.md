# 🚚 Strapi 物流行业门户模板

## 📋 项目介绍
这是一个基于Strapi v4的物流行业门户网站模板，提供完整的物流企业网站解决方案。集成运输管理、仓储管理、配送服务、物流追踪等核心功能，支持快速构建专业的物流门户网站。

## 🚀 快速开始
### 环境要求
\`\`\`bash
Node.js >= 16
PostgreSQL >= 10
Redis >= 6
\`\`\`

### 安装步骤
1. 克隆模板
\`\`\`bash
npx create-strapi-app my-logistics-site --template logistics
\`\`\`

2. 安装依赖
\`\`\`bash
cd my-logistics-site
yarn install
\`\`\`

3. 环境配置
\`\`\`bash
cp .env.example .env
# 配置数据库和其他设置
\`\`\`

4. 启动服务
\`\`\`bash
yarn develop
\`\`\`
