# 💇 Strapi 美容美发门户模板

## 📋 项目介绍
这是一个基于Strapi v4的美容美发行业门户网站模板，提供完整的美容美发企业网站解决方案。集成服务管理、预约管理、会员管理、营销管理等核心功能，支持快速构建专业的美容美发门户网站。

## 🚀 快速开始
### 环境要求
\`\`\`bash
Node.js >= 16
PostgreSQL >= 10
Redis >= 6
\`\`\`

### 安装步骤
1. 克隆模板
\`\`\`bash
npx create-strapi-app my-beauty-site --template beauty
\`\`\`

2. 安装依赖
\`\`\`bash
cd my-beauty-site
yarn install
\`\`\`

3. 环境配置
\`\`\`bash
cp .env.example .env
# 配置数据库和其他设置
\`\`\`

4. 启动服务
\`\`\`bash
yarn develop
\`\`\`
